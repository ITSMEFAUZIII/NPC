MOVEMENT:
  Walk speed  : 5.0 units/sec
  Run speed   : 9.0 units/sec (Shift held)
  Sprint speed: 13.0 units/sec (double-tap W, 3 sec duration)
  Jump height : 4.0 units
  Air control : 0.4 (reduced movement while airborne)
  Dodge roll  : Space while moving = 6 unit dash, 0.8s cooldown
                iframe (invincible) during first 0.3s of dodge

MOVEMENT IMPLEMENTATION:
  Direction from camera yaw + WASD input
  cannon-es Capsule body (radius 0.4, height 1.6)
  velocityY handled manually (gravity -20 units/sec²)
  Ground check: raycast downward 0.6 units
  Slope limit: if slope > 45deg, slide down

COMBAT STATE MACHINE:
  idle → attacking → recovery → idle
  idle → dodging → idle
  idle → casting → idle (for skills with cast time)
  Any state → stunned → recovery
  Any state → dead

COMBAT INPUTS:
  Left Click  : Basic Attack
  Right Click : Block/Parry (Swordsman) / Aim (Archer)
  Q           : Skill 1
  E           : Skill 2
  R           : Skill 3
  F           : Elemental Skill
  T / X       : Ultimate Skill
  Space       : Dodge Roll
  Z           : Lock-on target (cycle with mouse wheel)

LOCK-ON SYSTEM:
  Press Z: lock camera to nearest enemy within 15 units
  Camera orbits around locked enemy
  Movement is relative to locked direction
  Press Z again or enemy dies: release lock
  Visual: circular ring indicator around locked target

HITBOX SYSTEM:
  Player has attack hitbox (enabled during attack frames only)
  BoxGeometry or SphereGeometry per attack
  Check overlap with enemy colliders each frame during attack
  On hit: apply damage, trigger hit reaction on enemy

SECTION 6 — COMBAT SYSTEM (combat/CombatSystem.js)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DAMAGE FORMULA:
  baseDamage = skill.damage * player.atk
  elementBonus = player.stats.elementBonus[skill.element] || 0
  critMultiplier = isCrit ? (player.stats.critDmg / 100) : 1.0
  defReduction = enemy.def / (enemy.def + 100)
  finalDamage = baseDamage * (1 + elementBonus/100)
               * critMultiplier
               * (1 - defReduction)
               + random(-5%, +5%) variance

CRIT CALCULATION:
  roll = Math.random() * 100
  isCrit = roll < player.stats.crit
  On crit: damage number shows in gold color, larger font

DAMAGE NUMBERS (components/HUD/DamageNumbers.jsx):
  Float above hit position, fade + rise over 1.2s
  Normal hit  : white #ffffff, size 16
  Crit hit    : gold #ffd700, size 22, "!" suffix
  Element hit : element color, size 18
  Reaction    : element color, size 24, reaction name shown
  Heal        : green #40ff80, size 16, "+" prefix
  Max 20 damage numbers visible at once

STATUS EFFECTS (combat/StatusEffects.js):
  BURN     : Fire,    8% maxHP/sec, 4s, orange particles
  FREEZE   : Ice,     can't move 2s, blue crystal overlay
  PARALYZED: Thunder, can't act 1.5s, yellow sparks
  POISON   : Dark,    5% maxHP/sec, 8s, green drip
  STUNNED  : Various, can't act, stars over head
  BLIND    : Wind,    -80% accuracy, grey overlay
  CURSED   : Dark,    -30% all stats, purple sigil
  BURNING  : Fire+,   16% maxHP/sec, 3s (stronger burn)
  ROOTED   : Earth,   can't move, roots wrap feet

Stack rules:
  Same effect resets duration (no stacking damage)
  FREEZE overrides PARALYZED
  BURN + FREEZE = STEAM BURST (clears both, AoE)

ENEMY HIT REACTION:
  Light hit (< 20% HP): brief flinch animation
  Medium hit (< 40% HP): stagger back 0.5 units
  Heavy hit (> 40% HP): full knockback 2 units
  Kill hit: ragdoll-like death animation (scripted)

SECTION 7 — ENEMY AI (characters/enemies/EnemyAI.js)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ENEMY STATE MACHINE:
  IDLE     → patrol waypoints, random idle animations
  ALERT    → player detected, turn toward player
  CHASE    → move toward player at chaseSpeed
  ATTACK   → in range, execute attack pattern
  STAGGER  → hit reaction, brief vulnerability
  RETREAT  → low HP enemies may flee briefly
  DEAD     → death animation, loot drop, dispose

DETECTION:
  Sight range: 15 units (varies per enemy)
  Sight angle: 120 degrees forward cone
  Sound range: 8 units (player running triggers this)
  Line of sight: raycast to player, blocked by terrain
  Once alerted: stays alerted for 10s after losing sight

PATHFINDING:
  Simple waypoint system (not full NavMesh)
  Waypoints placed manually per area
  Enemy picks nearest waypoint toward player
  Obstacle avoidance: if blocked, random left/right deflection
  Flying enemies: direct path to player, ignore terrain

ATTACK PATTERNS (per enemy type in data/enemies.js):
  Each enemy has array of attacks with:
    name, damage, range, cooldown, telegraphTime, animDuration
  telegraphTime: visual warning before attack lands
    (red indicator circle on ground for AoE, charge animation)
  Enemy AI picks attack based on distance and cooldown

AGGRO SYSTEM:
  Single player: always target player
  Multiple players (future): threat table
  Taunt skill (Swordsman): force target player for 5s

ENEMY VISUAL DESIGN:
  All procedural geometry
  Health bar above enemy (hidden until player attacks)
  Element indicator icon above health bar
  Aggro state: eyes change color (yellow = alert, red = attack)

  Phase 3 trigger: 30% HP (final bosses only)

SECTION 9 — ROLE SWITCH SYSTEM

SWITCH LOCATIONS:
  - Role Shrine in each area (glowing stone pillar)
  - Camp/checkpoint (press C to open camp menu)
  - NOT during combat (enforced 5s out-of-combat requirement)

WHAT CARRIES OVER:
  - Player level + EXP
  - Gold + inventory items + materials
  - Quest progress
  - World state (defeated enemies, opened chests)

WHAT CHANGES:
  - Visual model completely swaps
  - All stats reset to role base + level bonus
  - Skills completely replaced
  - Equipment slots reset (old equipment stored)
  - New role equipment auto-equips from stored inventory
  - HP/MP refilled on role switch

TRANSITION ANIMATION:
  Brief white flash (0.5s)
  Player model dissolves into particles matching new role color
  New model assembles from particles
  Role name displayed: "MAGICIAN" (large, centered, fades)
  New role music motif plays briefly
